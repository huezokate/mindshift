## Context

Define the MindShift design system as CSS custom properties in code and as a Figma reference frame.

Use the old UI (presentation slides) as color/mood inspiration — not literally. Extract the depth, warmth, and intentionality of that palette and translate it into a token set that works for the canvas mind map.

## Acceptance Criteria

- CSS `--token` variables defined in `:root` for all node colors (bg + fg per category), canvas background, hub styling, typography scale, spacing, and arrow stroke
- All hardcoded hex/px values in node CSS replaced with token references
- A "Design System" frame in the Figma file showing: color swatches, type scale, and node specimens using the new tokens
- No visual regressions on page 3
