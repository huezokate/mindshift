## Vision

Build MindShift V200 — a Next.js 16 app in `/V200/` that connects Clerk (auth), Supabase (DB), Stripe (payments), Resend (email), and Claude AI (responses). The goal is a trial prototype to validate traction: beautiful, interactive UI as close to the Figma designs (node 414:4628) as possible.

**Tech stack:** Next.js 16 App Router · TypeScript · Tailwind v4 · Clerk v7 · Supabase v2 · Stripe v22 · Resend v6 · Anthropic SDK · Framer Motion

**Design language:** Cyberpunk theme — dark bg (#080810), neon cyan/pink/violet/green glows, scanline overlay, monospace font (Courier New / Alumni Sans SC), glassmorphism cards.

**App routes:**
- `/` — Landing page (public, marketing)
- `/sign-in` / `/sign-up` — Clerk auth (cyberpunk styled)
- `/app/onboarding` — Vision wizard (5-step: vent → category goals → figure pick)
- `/app/mindmap` — Interactive spatial mind map canvas (7 category nodes)
- `/app/lens` — Figure/lens selection grid (9 figures)
- `/app/response` — AI response view (terminal aesthetic, share CTA)

**API routes:**
- `POST /api/generate-response` — Claude API call (authenticated)
- `POST /api/send-email` — Resend email (authenticated)
- `POST /api/webhook/stripe` — Stripe webhook handler
