## Context

Define the complete CSS custom-property token set for MindShift's fused aesthetic. Extends the current cyberpunk tokens in mindshift-cyberpunk.html with:
- Kawaii accent palette (soft pastels that complement cyan/pink)
- Notepad texture tokens (opacity, blend-mode hints)
- Fluid typography scale using clamp() for mobile→web
- Spacing scale (8px base grid, fluid at breakpoints)
- Border-radius tokens (cyberpunk = 0-2px, kawaii = 8-16px for specific contexts)
- Animation duration + easing tokens
- Breakpoint tokens for the responsive system

## Acceptance Criteria

- Complete `:root` token block ready to drop into HTML
- Each token documented with: name, value, usage context
- Responsive fluid values for type and spacing
- No hardcoded values in component CSS — everything references a token
