## Context

Clicking a category node should expand it to show full detail: all goal bullet points (some are truncated in the collapsed view), plus an editable text area so the user can add or refine their goals.

This replaces the old "deep dive" sections from the linear UI. The expansion should feel like the node grows in place, not a page transition.

## Acceptance Criteria

- Clicking a node toggles an expanded state
- Expanded state shows all goal bullets for that category (not truncated)
- Expanded state includes a text area for adding/editing goals inline
- Changes persist in memory (survive navigation away and back within the session)
- Clicking outside the expanded node, or pressing Escape, collapses it
- Only one node can be expanded at a time (opening a second collapses the first)
- Expanded node sits above other nodes (z-index)
- Expansion animation is smooth (CSS transition, not instant)
