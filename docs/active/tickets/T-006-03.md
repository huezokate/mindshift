## Context

Design the visual language that fuses three aesthetic registers:

1. **Cyberpunk** (current) — dark backgrounds, neon cyan/pink, scanlines, monospace, glow effects
2. **Kawaii** — soft rounded forms, pastel accent moments, playful micro-animations, expressive character moments, cute empty states
3. **Notepad/analog** — ruled lines, handwritten-feel type, paper texture moments, ink-bleed effects, scribble animations

These must coexist without visual noise. The cyberpunk skeleton holds; kawaii + notepad are emotional seasoning applied at key engagement moments (onboarding, empty states, celebration, AI response reveal).

Consider:
- Which screens/moments get which register
- How transitions signal register shifts
- Color palette extensions (which kawaii pastels play with the cyan/pink without clashing)
- Typography pairing (mono + a kawaii/handwritten display face)
- Animation vocabulary per register

## Acceptance Criteria

- `research.md` with moodboard references, screen-by-screen aesthetic assignment, color palette extension, font pairing candidates
- `design.md` with the chosen fusion approach, rejected alternatives, and rationale
