## Context

Build 3 Wizard/Vision Input screens (Cyberpunk, Kawaii, Notepad). This is the core interaction screen — where users type their "vent" / current situation and the AI will respond through the lens of a historical figure.

Screen content: status bar, screen title/prompt ("what's on your mind?" / "tell me what's happening ✦" / "write it down:"), large multi-line text input area, character hint / helper text, progress indicator (step 1/3 or similar), CTA to proceed, UI theme switcher.

## Acceptance Criteria

- 3 frames (390×844) with correct theme styling
- Text input area fills most of the screen, styled per theme (terminal-style dark / rounded kawaii card / ruled notepad lines)
- Prompt text uses theme fonts and tone
- CTA button to proceed to lens selection
- UI theme switcher present and showing active theme
