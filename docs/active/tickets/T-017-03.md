## Figure card (LensPreviewCard)

Each card (120px wide):
- White bg, `var(--fig-border)` (2px solid #400b14 in kawaii), `var(--fig-radius)` (32px), padding 10px
- **Avatar image** — 76px circle, `var(--fig-avatar-border)` (pink border), `var(--fig-avatar-shadow)`, gradient bg (`var(--fig-avatar-grad)`), `<img src={getFigureImg(figure, theme)} />`
- **Name** — Bold 12px, `var(--fig-name-unsel)` (#7e2091 in kawaii), uppercase, centered
- **Descriptor** — Regular 10px, `var(--fig-desc)` (#9490b8 in kawaii), uppercase, centered

**Selected state:**
- Border changes to `var(--fig-border-sel)`
- Name color changes to `var(--fig-name-sel)`
- Add `var(--fig-shadow-sel)` box-shadow
