## Context

Build 3 Mind Map Canvas screens (Cyberpunk, Kawaii, Notepad). This is the main visualization — an interactive spatial canvas with a central hub ("In 5 years…") connected to 7 organic-shaped category nodes: Career, Creativity, Health & Wellness, Relationships, Travel, Finances, Scandinavia.

Screen content: status bar, spatial canvas background, center hub node, 7 category nodes connected by curved arrows, each node shows category label + 2-3 goal bullets (placeholder), mood board image thumbnail. UI theme switcher visible as a floating element.

Design reference: Figma section "Lo-fi new UI for mind map screen" (node 19-99).

## Acceptance Criteria

- 3 frames (390×844) with spatial canvas feel
- Central hub node with "In 5 years…" visible and centered
- 7 surrounding category nodes placed organically (not grid-aligned)
- Curved connections between hub and nodes
- Each theme applies its visual language: dark neon glow / soft pastel bubbles / sketchy notepad nodes
- UI theme switcher as a floating element at top or bottom
