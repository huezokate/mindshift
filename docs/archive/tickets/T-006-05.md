## Context

Produce a WCAG 2.1 AA compliance spec and user engagement pattern catalogue for MindShift.

**WCAG scope:**
- Contrast ratios: cyberpunk neon on dark backgrounds (cyan #00FFC8 on #07070F passes — audit all text/UI combos)
- Focus indicators: visible focus rings that work with the neon aesthetic
- Motion: respect prefers-reduced-motion for scanlines, glows, and animations
- Touch targets: 44×44px minimum on mobile
- Screen reader: semantic HTML, ARIA roles for the wizard flow and mindmap canvas
- Keyboard navigation: full tab order through all 5 screens

**Engagement patterns:**
- Onboarding delight (first-run experience, empty state)
- Progress feedback (wizard progress indicator, AI loading states)
- Micro-interactions (button press, card select, node expand)
- Celebration moments (mindmap complete, goal saved)
- Retention hooks (save/share CTA placement, re-engagement prompts)
- Social proof / inspiration moments (figure quotes, perspective reveals)

## Acceptance Criteria

- WCAG audit table: component, issue, fix, priority
- Contrast ratio table for all color pairs in use
- Engagement pattern catalogue with: pattern name, trigger, implementation hint, screen it applies to
- prefers-reduced-motion strategy
