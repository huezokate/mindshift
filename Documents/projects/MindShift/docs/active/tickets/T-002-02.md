## Context

The current node implementation has a structural bug: clip-path is applied to the entire .canvas-node div, which clips text content along with the background. Triangle and diamond nodes visibly cut off labels and goal text.

Fix this by separating the shape layer (::before pseudo-element with clip-path) from the text layer (direct children, unclipped). Also apply the design tokens from T-002-01 to achieve the improved visual quality.

## Acceptance Criteria

- No text content clipped by any node shape (all 7 categories)
- ::before pseudo-element carries clip-path and background color
- Direct children of .canvas-node carry z-index: 1 to render above pseudo-element
- Typography uses token-based font sizes and weights
- Color classes use CSS custom property references
- Hover state still works (scale + drop-shadow)
- Expand state (cn-expanded, T-001-05) still works
- All 7 nodes visually distinct and coherent as a set
