## Context

Seven satellite nodes orbit the central hub, each representing a life category. In the Figma lo-fi, each node uses a different organic SVG shape:

| Category | Shape |
|---|---|
| Career | Ellipse |
| Creativity | Triangle (regular polygon) |
| Health & Wellness | Diamond (rotated square) |
| Relationships | Irregular blob/triangle |
| Travel | Pentagon/irregular blob |
| Finances | Rounded rectangle |
| Living Situation | Tall rounded rectangle (with photo grid) |

Each node contains:
- Category label (e.g., "career", "creativity")
- A header aspiration line (e.g., "I work...", "I publish my novel")
- 2-5 bullet point goals for that category
- Space for mood images (implemented in T-001-06)

Nodes are pre-positioned around the center in a roughly clock-face layout matching the lo-fi. Their positions are hardcoded world coordinates (not computed dynamically).

## Acceptance Criteria

- All 7 category nodes render at correct world positions around the center hub
- Each node uses its designated organic SVG shape as a clip/border
- Each node shows: category label, header line, and bullet goals
- Goal content comes from the user's onboarding answers where applicable; remaining categories use sensible placeholder text
- Nodes have a hover state (subtle highlight or scale)
- Each node is clickable (click handler wired up, detail expansion handled in T-001-05)
- The layout looks correct at 1x zoom with the canvas centered on the hub
