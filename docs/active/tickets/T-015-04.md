## Context

Build the interactive mind map canvas at `/app/mindmap`. This is the visual centrepiece — 7 organic-shaped category nodes arranged around a central hub on an infinite canvas. Reference: V100's mind map implementation and Figma node 414:4628.

## Canvas Layout

- Central hub: "In 5 years..." (white glow, pulsing)
- 7 nodes radially positioned: career, creativity, health, relationships, travel, finances, living
- Each node: glassmorphism card with category icon, label, 3 goal bullets
- Curved SVG arrows connecting hub to nodes
- Node colors from tokens.css (`--node-career-bg`, etc.)

## Interactions

- Pan: drag canvas
- Zoom: pinch or scroll wheel
- Tap node: expand to show all goals + "ASK A LENS" button
- "ASK A LENS" → navigates to `/app/lens` with node category in query
- Framer Motion layout animations for node expand/collapse

## Implementation Notes

- Use `<svg>` for arrows (not a canvas library — keep it simple for MVP)
- Node positions are fixed (not drag-to-reposition in MVP — that's beta)
- Load vision entry from Supabase by entry ID from session/params
- If no entry exists, redirect to `/app/onboarding`

## Acceptance Criteria

- All 7 nodes visible, positioned radially, connected to hub with curves
- Pan works on mobile (touch) and desktop (mouse drag)
- Zoom works (pinch + scroll)
- Node tap expands with goals visible
- "ASK A LENS" button on expanded node works
- Cyberpunk visual style: dark bg, neon node borders, glow on hover
