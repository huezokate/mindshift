## Context

Build 3 AI Response screens (Cyberpunk, Kawaii, Notepad). After selecting a lens, users see the historical figure's perspective on their situation. This is the payoff screen — it should feel like a real transmission from that figure.

Screen content: status bar, figure portrait (larger, header area), figure name + "via [theme] lens" label, response text block (multi-paragraph), action row: Save to Journal, New Lens, Share. UI theme switcher.

Theme-specific treatments:
- Cyberpunk: terminal-style text reveal, scanlines, green-on-dark response text, "TRANSMISSION FROM [NAME]" header
- Kawaii: speech bubble or card with soft shadow, figure portrait in circle, "✨ [Name] says…" heading
- Notepad: handwriting-style response block with ruled lines, stamped "PERSPECTIVE" label, figure portrait as polaroid

## Acceptance Criteria

- 3 frames (390×844) with figure portrait, response text, and action row
- Response text is placeholder but styled correctly per theme
- Action row (Save / New Lens / Share) styled per theme
- Figure portrait uses theme-appropriate image
- UI theme switcher present
