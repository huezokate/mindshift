## Vision

Complete the core MVP loop: vent → apply lens → save → journal.

After the AI response renders, the user hits Save and the vent+response is persisted to Supabase. The journal route shows all saved entries — each vent session as a card with the lens responses beneath it. Multiple lenses can be applied to the same session. Everything is React components (no image generation).

**Schema:** Two-table normalized design — `vent_sessions` (one row per vent) + `lens_responses` (one row per lens applied). Journal groups by session.

**Out of scope for this story:** image export, stickers, watermark branding, sharing features.
