## Context

The canvas centers on a hub node showing the user's 5-year vision. In the lo-fi Figma, this is a text block that reads "In 5 years..." with a subtitle listing the life categories: "career, personal development, relationships, health and wellness, travel, finances, and creativity."

This node is generated from the user's onboarding answers (questions 1-3 from the existing flow). It sits at world coordinate (0, 0) and all satellite nodes orbit around it.

## Acceptance Criteria

- Central node renders at the canvas center on load
- Displays the user's "where you want to be" text (from q2 in the existing form)
- Displays a subtitle listing the life categories that have satellite nodes
- Node has a distinct visual style (larger, different shape or weight) compared to satellites
- Clicking the central node does nothing (it's informational, not interactive)
- Node text wraps cleanly at a max width; no overflow
- If no user data exists yet (direct navigation), node shows placeholder text
