## Context

Build the lens/figure selection screen at `/app/lens`. 9 historical figures shown in a grid — user picks one to get their AI perspective on the selected mind map category.

## Figures (from existing V100 data)

Use the same 9 figures from mindshift.html. Each has: id, name, era descriptor, system prompt, cyberpunk portrait path.

## Acceptance Criteria

- 3×3 grid of figure cards (portrait image, name, era)
- Tap to select (highlighted state with neon border)
- Selected figure shows description tooltip/card
- "GET PERSPECTIVE" button → calls `/api/generate-response` with figure + vent prompt
- Loading state: terminal typing animation
- On response received: navigate to `/app/response`
- Back button → returns to `/app/mindmap`

## Components

- `src/components/ui/FigureCard.tsx` — individual figure portrait card with hover/active states
- Page: `src/app/app/lens/page.tsx`
