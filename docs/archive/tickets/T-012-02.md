## Scope

All edits land in `V100/V100.html`.

### 1. Horizontal pill theme selector
Replace `.theme-bar` (fixed 3-tab flex) with `.theme-picker` (horizontal scroll pill row). Pill style per theme. Supports future themes by scrolling.

### 2. Lens screen → horizontal carousel
Replace `.figure-grid` (3-col grid) with `.lens-carousel` (horizontal scroll strip of avatar chips, snap scrolling). Tapping a chip opens the preview overlay.

### 3. Preview popup → embedded carousel
Add the lens carousel strip INSIDE the preview overlay at the bottom. Swiping/tapping a chip in the strip updates the preview above. Active chip highlighted. Replaces needing to close popup to browse.

### 4. Preview popup CTA personalisation
"Select the Lens" → "Hear from [Name]" — dynamically updated when preview opens.

### 5. Daily lens limit
Replace `lensUseCount` (session counter) with localStorage `{ date, count }`. Resets at midnight. Limit: 3 free per day. Gate copy updated per research.

### 6. Account form simplification
Remove phone + password fields. Add "confirm email" field. Add privacy note: "We won't spam you — only new feature announcements, ~once a month."

### 7. CTA copy
Apply all copy changes from T-012-01 research (THEME_COPY + static HTML labels).

### 8. Share preview screen (stub)
New `screen-share` stub: shows the response as a card with a MindShift watermark overlay + social icons. Full social share implementation is out of scope; stub navigates there from "Share it" button.

### 9. Post-share success toast
After sharing, show a 5s auto-dismiss toast: "Shared ✓ — keep it in your journal by creating an account →" with a tap-to-go-to-account action.

## Acceptance Criteria
- Theme picker scrolls horizontally, active pill highlighted per theme
- Lens screen shows horizontal avatar chip row, no grid
- Popup preview has carousel at bottom; swiping updates preview content
- CTA reads "Hear from [Name]" in all 3 themes
- Lens limit persists across page refresh via localStorage, resets daily
- Account form has 3 fields: nickname, email, confirm email
- All copy matches T-012-01 table
