## Context

Analyze the MindShift Figma file (https://www.figma.com/design/Mubv0Ghdm2SPxF42JVsX8M/MindShift?node-id=270-1180) to extract:
- All lofi screen layouts and the interaction flows they imply
- Any design system tokens, components, or style references already defined
- The spatial/canvas mindmap screen structure and node anatomy
- Implicit UX patterns (spacing rhythm, hierarchy, information density)

The lofi screens are the primary layout authority — implementation must be faithful to them while elevating the visual language to cyberpunk + kawaii.

## Acceptance Criteria

- `research.md` cataloguing each lofi screen with: purpose, key components, layout grid, interaction hints
- Component inventory: what repeats, what is screen-specific
- Token candidates: spacing rhythms, type sizes, color roles identified from the lofi
- Open questions for the aesthetic-fusion ticket to resolve
