## Context

Build the public landing page at `/` in `/V200/src/app/page.tsx`. This is the first impression — must be visually stunning and match the Figma cyberpunk aesthetic. Reference: Figma node 414:4628 (MindShift design file).

Design language reference (from mindshift.html):
- Background: #080810 with scanline overlay (already in globals.css)
- Hero: large Alumni Sans SC heading, neon cyan glow text
- CTA button: asymmetric border-radius (4px 1px 1px 4px), cyan border with hover glow
- Typography: Courier New mono for body, Alumni Sans SC for headers
- Glassmorphism cards: rgba(13,13,26,0.7) + backdrop-blur + rgba(0,245,255,0.15) border

## Acceptance Criteria

- `/` renders a full cyberpunk-styled landing page with:
  - Hero section: "MINDSHIFT" logo (Alumni Sans SC, cyan glow) + tagline + CTA "ENTER" button → `/sign-up`
  - Features section: 3 cards explaining the app (mind map, AI lenses, perspectives)
  - Pricing teaser: free vs pro pill
  - Footer: minimal, monospace
- Framer Motion entrance animations (fadeSlideUp stagger)
- Fully responsive (mobile-first)
- Clerk `<SignedIn>` / `<SignedOut>` to show "OPEN APP" vs "GET STARTED" CTA
- No hardcoded inline styles — use CSS custom properties from tokens.css

## Key Components to Create

- `src/components/layout/Navbar.tsx` — logo + nav links + auth button
- `src/components/ui/GlowButton.tsx` — reusable cyberpunk button (primary/secondary variants)
- `src/components/ui/GlassCard.tsx` — reusable glassmorphism card
- `src/app/page.tsx` — landing page assembly
