## Context

In the Figma lo-fi, each satellite node is connected to the central hub by a curved arrow (hand-drawn aesthetic). These are rendered in the SVG overlay layer established in T-001-01.

The arrows should:
- Start from the edge of each satellite node (not the center)
- End at the edge of the central hub
- Use a gentle cubic bezier curve, not a straight line
- Have an arrowhead at the hub end
- Feel slightly organic/imperfect to match the lo-fi aesthetic

## Acceptance Criteria

- One curved SVG arrow per satellite node (7 total)
- Arrows connect node edge → hub edge, not center → center
- Arrows use cubic bezier curves
- Arrowhead rendered at the hub end
- Arrows update their paths correctly when zoom/pan changes (they live in world-space SVG coordinates)
- Arrows do not overlap node content
- Arrow stroke style matches the lo-fi (thin, dark, slightly casual)
