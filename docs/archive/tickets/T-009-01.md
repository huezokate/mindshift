## Context

Research the existing MindShift codebase, existing Figma designs, and all 3 design systems to produce a complete interaction map for the 5 screens × 3 themes. Identify: what content lives on each screen, what user actions are possible, what transitions connect screens, and how the theme toggle changes things visually vs. structurally.

## Acceptance Criteria

- `research.md` enumerates all 5 screens with their content inventory
- Documents all user interactions per screen (tap targets, inputs, navigation)
- Documents theme-toggle behavior: what changes per theme on each screen
- References existing code (mindshift.html) and Figma node IDs where relevant
- Identifies what portrait assets are available per theme (cyberpunk/kawaii/notepad × 15 figures)
