## Context

Build the AI response view at `/app/response`. Shows the chosen figure's perspective with a terminal-style reveal animation. Must feel premium and shareable.

## Acceptance Criteria

- Figure portrait (large, left column on desktop / top on mobile)
- Response text appears with typewriter animation (char-by-char or line-by-line)
- "SAVE TO JOURNAL" → saves response to Supabase `responses` table
- "SHARE" → generates a styled image card (canvas API or OG image route) + native share sheet
- "TRY ANOTHER LENS" → back to `/app/lens`
- "BACK TO MAP" → back to `/app/mindmap`
- Subtle Stripe CTA: "Unlock unlimited lenses — Go Pro" (only shown to free users after 3 uses)

## Components

- `src/components/ui/TypewriterText.tsx` — animated text reveal
- `src/components/ui/ShareCard.tsx` — shareable image preview
- Page: `src/app/app/response/page.tsx`
