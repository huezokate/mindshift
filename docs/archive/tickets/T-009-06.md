## Context

Build 3 Lens Selection screens (Cyberpunk, Kawaii, Notepad). Users pick a historical figure to get their perspective through. The 9 available figures are displayed in a 3×3 grid with portrait, name, and era.

Figures (9 total): Ada Lovelace, Albert Einstein, Frida Kahlo, Nikola Tesla, Marie Curie, Leonardo da Vinci, Harriet Tubman, Alan Turing, Cleopatra.

Each figure has a portrait image in 3 styles (portraits/cyberpunk/, portraits/kawaii/, portraits/notepad/).

Screen content: status bar, section heading ("select your lens" / "choose your guide ✦" / "pick your perspective:"), 3×3 portrait grid with name + era labels, selected state (highlighted border/glow/underline per theme), back navigation, confirm button.

## Acceptance Criteria

- 3 frames (390×844) with 3×3 portrait grid
- Portrait images use theme-appropriate style (cyberpunk/kawaii/notepad PNGs)
- One figure shown in active/selected state per frame
- Name + era labels styled per theme fonts
- Active selection uses theme-appropriate highlight (cyan glow / pink border / navy underline)
- UI theme switcher present
