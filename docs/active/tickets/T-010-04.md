## Context

The Figma response screen has a specific action bar: Save (icon + label), Decorate (icon + label), Socials (icon + label) in a row below the response. The bottom nav has: Try another Lens | Home | Continue conversation. Also: Figma sticky notes specify that the 4th lens attempt in a session should trigger the account creation gate.

## Acceptance Criteria

- `research.md` maps current response screen actions vs. Figma spec
- `design.md` specifies action bar layout and account gate trigger logic
- Implementation: response action row shows Save / Decorate / Socials with icons
- Implementation: bottom nav shows correct labels (Try another Lens, New, Continue conversation)
- Implementation: `lensUseCount` tracks how many lenses have been used; on 4th tap of any figure card, show account screen first
- Account gate has a "Skip for now" path that proceeds to lens without account creation
- Gate only triggers once per session (sessionStorage flag)
